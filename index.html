Personal Clip Publisher

A single-user desktop tool for publishing my own short-form videos to my own accounts.

What it does

Personal Clip Publisher is a private tool I run on my own computer. I prepare finished vertical (1080×1920) short videos and their captions, and the tool uploads them directly to my own social accounts through each platform's official API — instead of manually transferring every file to my phone to post.

How it works

  • I sign in once to my own account via the platform's official OAuth flow.
  • I prepare a finished MP4 and a caption on my computer.
  • The tool uploads that video to my own account via the official Content Posting API.
  • The video appears on my profile; I review performance in the normal app.

Scope & privacy

This is a personal, single-user tool. It posts only to the one account that authorized it, handles only my own access token (stored locally on my computer), and never collects, aggregates, or processes any other person's content or data.
